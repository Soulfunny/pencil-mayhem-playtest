# PENCIL MAYHEM — Web Prototype Playtest

**DRAW. FLICK. WRECK.**

Public M1 playtest for the 2D hand-drawn pencil-flick battle game.

Play: <https://soulfunny.github.io/pencil-mayhem-playtest/>

## Controls

1. Select any tank belonging to the current player.
2. Press near the tank and pull the pencil opposite the intended travel direction.
3. Crossing the trigger threshold fires automatically.
4. A stroke that crosses an enemy tank destroys it. Eliminate all three enemy tanks to win.

Use the in-game **Stable / Real / Naughty** presets to compare flick feel.

## Playtest scope

This is prototype `M1-20260715`. It tests only the core 3-vs-3 pencil-flick movement, collision, turns and win conditions. Final art, traps, destructible objects, Doodle Garage, roguelite progression and online multiplayer are not included.

The M1 build has no account system, analytics or online save. The repository intentionally contains only the static deployment bundle and deployment checks; development source and design documents are not published here.
